Hexo 终端操作指南

发表于 2026-05-14 分类于 博客维护

这篇文章记录我维护 Hexo 博客时最常用的终端命令，默认博客目录是 E:\blog。

进入博客目录

后面的命令都建议在这个目录里执行。

如果改了文章或主题，可以刷新浏览器看效果。预览结束时，在终端按 Ctrl + C 停止服务。

新建文章

Hexo 会在 source\_posts 里生成一个 Markdown 文件。文章开头建议保留这样的头部信息：

title 决定网页标题，date 决定排序时间，categories 和 tags 用来归档。

清理并生成静态网页

npm run clean 会删除旧的生成结果，npm run build 会把 Markdown 文章生成到 public 文件夹。发布前建议总是先执行这两步。
